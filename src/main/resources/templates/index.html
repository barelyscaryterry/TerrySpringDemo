About This Project

This project demonstrates a simple Spring Boot application with:

  • Spring DevTools for automatic restarts
  • Lombok for reducing boilerplate code
  • JUnit for testing
  • RESTful endpoints
  • Thymeleaf templating with fragments
  • Responsive navigation bar
